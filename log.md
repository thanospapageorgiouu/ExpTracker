# Windows Expense Tracker App – Personal Log

## Σκοπός
Η εφαρμογή στοχεύει να παρακολουθεί **ώρες εργασίας και εισόδημα** για προσωπική χρήση, με μελλοντική δυνατότητα **GUI** και **καθαρής αποθήκευσης δεδομένων σε JSON**.

---

## Βήματα & Καταγραφές

### [1] Επιλογή τεχνολογίας
- Γλώσσα: **C#** για native Windows υποστήριξη.  
- Εναλλακτικά σκεφτήκαμε **C++**, αλλά προτιμήθηκε C# για ευκολία GUI & .NET.  

### [2] Βασικά χαρακτηριστικά
- Διαχείριση χρημάτων και εισοδήματος (ωρομίσθιο).  
- Υποστήριξη **user profiles** με βασικά στοιχεία.  
- Υπολογισμός μηνιαίου εισοδήματος.  
- Ιστορικό ωρών και εισοδήματος ανά μήνα.  
- Μελλοντική δυνατότητα **οθόνης αναφορών / γραφημάτων**.  

### [3] Σχεδιασμός data / αρχιτεκτονική
- Αρχικά text-based, με **μετατροπή σε GUI αργότερα**.  
- Αποθήκευση σε **JSON μέσω Gson** (Java version) ή **JSON.NET** (C#).  
- Backup αρχείου πριν οποιαδήποτε αλλαγή στα δεδομένα.  

### [4] Σημαντικές αποφάσεις / σημειώσεις
- Κάθε λειτουργία θα έχει **βήμα-βήμα οδηγίες** για δοκιμή.  
- Η εφαρμογή θα είναι **προσωπική**, αλλά σχεδιασμένη ώστε να μπορεί να επεκταθεί σε πολλαπλούς χρήστες ή περιόδους αναφοράς.  
- GUI επιλογές: αρχικά **Windows Forms ή WPF** για native C#.
